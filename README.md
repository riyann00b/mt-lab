# MT Labs Digital Operations Playbook

Documentation and configuration assets for Moti Taka’s self-hosted business operations stack.

## Overview

This repository captures the essentials for running Moti Taka infrastructure: ERPNext deployments, automation workflows, messaging bridges, and supporting services. Each top-level folder owns its own in-depth README—start there for setup guides, runbooks, and troubleshooting.

## Repository Structure

- `assets/` — Branding elements, print layouts, and other shared visuals.
- `docker-files/` — Containerfiles, Compose bundles, and environment templates per service.
- `docs/` — Narrative documentation, operational runbooks, and reference material.
- `n8n-workflows/` — Automation exports and playbooks connecting ERPNext with external channels.
- `scripts/` — Helper utilities that streamline maintenance and recurring tasks.
- `experimental/` — Work-in-progress ideas and prototypes awaiting promotion to production.
